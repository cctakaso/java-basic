Q01. ループ内の処理を最低一度は実行するループ構文はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. do-while
b. while
c. 拡張for文
d. すべてのループは最低1回実行される
-------------------------------------------------------------------------------

答え：
b
解説：
`do-while`ループは条件を最後に評価するため、最低1回は実行されます。

-------------------------------------------------------------------------------
Q02. ループ処理を現在のイテレーションで中断し、次のイテレーションに直ちに進むために
使用するキーワードは何ですか？  次の４つから正しい答えを選びなさい。
選択肢：
a. break
b. return
c. skip
d. continue
-------------------------------------------------------------------------------

答え：
d
解説：
`continue`文は、ループ内の残りの処理をスキップし、ループの次の繰り返し処理を開始します。

-------------------------------------------------------------------------------
Q03. `for`ループを使用して、0から9までの整数をコンソールに出力するコードを記述しなさい。
-------------------------------------------------------------------------------

答え：
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```
解説：
`for (初期化; 条件式; 更新式)`の構文を使用します。`int i = 0;`でカウンター変数を初期化し、
`i < 10;`が`true`である間ループを続け、各ループの終わりに`i++`でカウンターをインクリメント
します。

-------------------------------------------------------------------------------
Q04. `String`型の配列`fruits`の各要素を、拡張`for`文を使って表示するコードを記述しなさい。
-------------------------------------------------------------------------------

答え：
```java
String[] fruits = {"apple", "banana", "orange"};
for (String fruit : fruits) {
    System.out.println(fruit);
}
```
解説：
拡張`for`文は `for (型 変数名 : 配列またはコレクション)` の形式で記述し、
配列などの各要素を順番に取り出して処理するのに便利です。
インデックス管理が不要で、コードが簡潔になります。

-------------------------------------------------------------------------------
Q05. `while`ループを使って、20以下の整数のうち3の倍数以外の奇数を出力するコードを記述しなさい。
-------------------------------------------------------------------------------

答え：
```java
int i = 0;
while (i++ < 20) {
    if (i % 2 == 1 && i % 3 != 0) {
        System.out.println(i);
    }
}
```
解説：
他にもコーディング方法はありますが、いずれにせよ条件によってiのインクリメントが
うまく実行できない場合があるのでタイミングと回数を間違えないようにして下さい。
