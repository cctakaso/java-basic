Q01. Javaプログラムのエントリーポイントとなるメソッドは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. start()
b. main()
c. run()
d. execute()
-------------------------------------------------------------------------------

答え：
b
解説：
Javaプログラムを実行した際に、最初に自動的に呼び出される特別なメソッドが`main`メソッドです。

-------------------------------------------------------------------------------
Q02. `main`メソッドの正しいシグネチャはどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. public static void main(String[] args)
b. public void main(String[] args)
c. public static int main(String[] args)
d. private static void main(String[] args)
-------------------------------------------------------------------------------

答え：
a
解説：
`main`メソッドは、`public static void`として宣言し、`String`の配列を引数として受け取る必
要があります。

-------------------------------------------------------------------------------
Q03. Javaの文の終わりには、どの記号を付ける必要がありますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. .
_b. ,
c. ;
d. :
-------------------------------------------------------------------------------

答え：
c
解説：
Javaの各文はセミコロン(;)で終了する必要があります。

-------------------------------------------------------------------------------
Q04. 複数行にわたるコメントを記述する際に使用する記号はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. //
b. ##
c. /* ... */
d. <!-- ... -->
-------------------------------------------------------------------------------

答え：
c
解説：
`/*`で始まり`*/`で終わるブロックは、複数行にわたるコメントを記述するために使用されます。
