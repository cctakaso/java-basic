Q01. Javaプログラムのエントリーポイントとなるメソッドは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. start()
b. main()
c. run()
d. execute()

答え：
b
解説：
Javaプログラムを実行した際に、最初に自動的に呼び出される特別なメソッドが`main`メソッドです。

Q02. `main`メソッドの正しいシグネチャはどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. public static void main(String[] args)
b. public void main(String[] args)
c. public static int main(String[] args)
d. private static void main(String[] args)

答え：
a
解説：
`main`メソッドは、`public static void`として宣言し、`String`の配列を引数として受け取る必要があります。

Q03. キーボードからの入力を受け取るために使用するクラスは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. System
b. Console
c. Scanner
d. Input

答え：
c
解説：
`java.util.Scanner`クラスを使用することで、キーボードからの入力を簡単に受け取ることができます。

Q04. `Scanner`クラスを使用する際に、最初に行うべきことは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. `new Scanner()`を実行する
b. `import java.util.Scanner;`を記述する
c. `System.in`を有効にする
d. `scanner.close()`を呼び出す

答え：
b
解説：
`Scanner`クラスは`java.util`パッケージに含まれているため、使用する前に`import`文で読み込む必要があります。

Q05. キーボードから入力された名前と年齢を読み取り、"こんにちは、[名前]さん！ あなたは[年齢]歳なのですね。" と表示するプログラムを作成しなさい。
答え：
```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("あなたの名前を入力してください: ");
        String name = scanner.nextLine();

        System.out.print("あなたの年齢を入力してください: ");
        int age = scanner.nextInt();

        System.out.println("こんにちは、" + name + "さん！ あなたは" + age + "歳なのですね。");
    }
}
```
解説：
`Scanner`オブジェクトを作成し、`nextLine()`メソッドで文字列を、`nextInt()`メソッドで整数を読み込みます。その後、`+`演算子で文字列を連結して表示します。
