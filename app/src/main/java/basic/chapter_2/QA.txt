Q01. Javaの文の終わりには、どの記号を付ける必要がありますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. .
_b. ,
c. ;
d. :

答え：
c
解説：
Javaの各文はセミコロン(;)で終了する必要があります。

Q02. プログラムのソースコードにおいて、数値を直接記述したものを何と呼びますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. 変数
b. リテラル
c. 式
d. 演算子

答え：
b
解説：
リテラルは、ソースコード内で直接表現される値（数値、文字列など）です。

Q03. `int a = 1 + 2;` という文における `1 + 2` の部分は何と呼ばれますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. 演算子
b. 変数
c. 式
d. コメント

答え：
c
解説：
式（expression）は、値、変数、演算子、メソッド呼び出しなどを組み合わせて、単一の値を生成するものです。

Q04. 複数行にわたるコメントを記述する際に使用する記号はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. //
b. ##
c. /* ... */
d. <!-- ... -->

答え：
c
解説：
`/*`で始まり`*/`で終わるブロックは、複数行にわたるコメントを記述するために使用されます。

Q05. 整数型の変数`a`を宣言し、値`10`を代入し、その値をコンソールに出力するプログラムを作成しなさい。
答え：
```java
public class StatementExample {
    public static void main(String[] args) {
        int a;
        a = 10;
        System.out.println(a);
    }
}
```
解説：
`int a;`で整数型の変数を宣言し、`a = 10;`で値を代入します。`System.out.println(a);`で変数の値を出力します。宣言と代入は`int a = 10;`のように一行で書くこともできます。
