Q01. `if`文の条件式として評価される値の型は何でなければなりませんか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. int
b. String
c. boolean
d. Object

答え：
c
解説：
`if`文の`()`内には、評価結果が`true`または`false`になる`boolean`型の式を記述する必要があります。

Q02. `switch`文で、各`case`ブロックの最後に`break`文を記述しなかった場合、どのような動作になりますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. コンパイルエラーになる。
b. `switch`文全体がスキップされる。
c. 一致した`case`の次の`case`ブロックも続けて実行される（フォールスルー）。
d. `default`ブロックのみが実行される。

答え：
c
解説：
`break`を省略すると、プログラムの実行は次の`case`ラベルに「落下」し、`break`に遭遇するか`switch`ブロックの終わりまで、後続の`case`の処理を実行し続けます。これをフォールスルーと呼びます。

Q03. ループ内の処理を一度も実行しない可能性があるループ構文はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. do-while
b. while
c. for-each
d. すべてのループは最低1回実行される

答え：
b
解説：
`while`ループは、ループの本体を実行する前に条件式を評価します。最初の評価で条件が`false`だった場合、ループ本体は一度も実行されません。一方、`do-while`ループは条件を最後に評価するため、最低1回は実行されます。

Q04. ループ処理を現在のイテレーションで中断し、次のイテレーションに直ちに進むために使用するキーワードは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. break
b. return
c. skip
d. continue

答え：
d
解説：
`continue`文は、ループ内の残りの処理をスキップし、ループの次の繰り返し処理を開始します。

Q05. `for`ループを使用して、0から9までの整数をコンソールに出力するコードを記述しなさい。
答え：
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```
解説：
`for (初期化; 条件式; 更新式)`の構文を使用します。`int i = 0;`でカウンター変数を初期化し、`i < 10;`が`true`である間ループを続け、各ループの終わりに`i++`でカウンターをインクリメントします。

Q06. `String`型の配列`fruits`の各要素を、拡張`for`文（for-eachループ）を使って表示するコードを記述しなさい。
答え：
```java
String[] fruits = {"apple", "banana", "orange"};
for (String fruit : fruits) {
    System.out.println(fruit);
}
```
解説：
拡張`for`文は `for (型 変数名 : 配列またはコレクション)` の形式で記述し、配列などの各要素を順番に取り出して処理するのに便利です。インデックス管理が不要で、コードが簡潔になります。

Q07. `while`ループを使って、`i`が5になったらループを終了するコードを記述しなさい。ループ内では`i`の値を出力し、`i`を1ずつインクリメントすること。
答え：
```java
int i = 0;
while (i < 10) { // 条件はループが継続する条件
    if (i == 5) {
        break; // iが5のときにループを抜ける
    }
    System.out.println(i);
    i++;
}
```
解説：
`break`文を使用すると、`if`文などの条件と組み合わせて、ループの継続条件が`true`であっても、任意のタイミングでループを強制的に終了させることができます。この例では0から4までが出力されます。
