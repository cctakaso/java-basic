Q01. Javaプログラムを実行するために必要な開発ツールは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. JRE
b. JDK
c. JVM
d. JDI
-------------------------------------------------------------------------------

答え：
b
解説：
Javaプログラムを動かすには、JDK (Java Development Kit)という開発ツールが必要です。JDKには、コンパイラと実行環境が含まれています。

-------------------------------------------------------------------------------
Q02. ソースコードをコンパイルするために使用するコマンドは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. java
b. javac
c. javadoc
d. jar
-------------------------------------------------------------------------------

答え：
b
解説：
javacコマンドは、.javaファイルを.classファイルにコンパイルするために使用されます。

-------------------------------------------------------------------------------
Q03. コンパイルされたJavaプログラムを実行するために使用するコマンドは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. java
b. javac
c. jre
d. execute
-------------------------------------------------------------------------------

答え：
a
解説：
javaコマンドは、コンパイルされた.classファイルを実行するために使用されます。

-------------------------------------------------------------------------------
Q04. 統合開発環境（IDE）の主な利点は何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. コードの実行速度が速くなる
b. コンパイルエラーが発生しなくなる
c. エディタ、コンパイラ、デバッガが統合されており、開発効率が向上する
d. OSに依存しないプログラムを作成できる
-------------------------------------------------------------------------------

答え：
c
解説：
IDEは、開発に必要なツールを一つにまとめることで、プログラマが開発に集中できる環境を提供します。

-------------------------------------------------------------------------------
Q05. "Hello World"と表示するプログラムを作成しなさい。
-------------------------------------------------------------------------------
答え：
```java
public class App {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
解説：
`main`メソッド内に`System.out.println("Hello World");`と記述することで、コンソールに"Hello World"と出力できます。
