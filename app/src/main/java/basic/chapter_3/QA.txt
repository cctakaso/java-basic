Q01. Javaの基本データ型（プリミティブ型）に含まれないものはどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. int
b. double
c. String
d. boolean

答え：
c
解説：
`String`は、クラスとして定義されている参照型です。`int`, `double`, `boolean`は基本データ型です。

Q02. 3.14という浮動小数点数のリテラルを`float`型の変数に代入する正しい記述はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. float f = 3.14;
b. float f = 3.14d;
c. float f = 3.14f;
d. float f = (float)3.14L;

答え：
c
解説：
Javaでは、小数点を含むリテラルはデフォルトで`double`型として扱われます。`float`型の変数に代入するには、リテラルの末尾に`f`または`F`を付けて`float`型であることを明示する必要があります。

Q03. 一度値を代入したら変更できない変数を宣言するために使用するキーワードは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. static
b. final
c. const
d. fixed

答え：
b
解説：
`final`キーワードを付けて宣言された変数は定数となり、再代入が禁止されます。

Q04. `long`型の変数の値を`int`型に変換する（キャストする）正しい構文はどれですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. int i = myLong;
b. int i = (int)myLong;
c. int i = toInt(myLong);
d. int i = myLong.toInt();

答え：
b
解説：
大きなサイズのデータ型から小さなサイズのデータ型への変換（縮小変換）は、キャスト演算子`(変換先の型)`を明示的に記述する必要があります。ただし、値が変換先の型の範囲に収まらない場合、データが欠落する（オーバーフローする）可能性があります。

Q05. 文字列の中にダブルクォーテーション（"）を含めるために使用するエスケープシーケンスは何ですか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. \'
b. \"
c. \\
d. \n

答え：
b
解説：
`\"`と記述することで、文字列リテラル内にダブルクォーテーション文字そのものを含めることができます。

Q06. 5つの整数を格納できる配列を宣言し、初期化するコードを記述しなさい。
答え：
```java
// 方法1: 宣言と同時に初期化
int[] numbers = {10, 20, 30, 40, 50};

// 方法2: サイズを指定して配列オブジェクトを生成
int[] numbers2 = new int[5];
```
解説：
配列は`型[] 配列名`で宣言します。`{}`を使って初期値を指定する方法と、`new`キーワードを使って指定したサイズの配列オブジェクトを生成する方法があります。

Q07. `int`型の配列`myArray`の要素数が10であるとき、`myArray[10]`にアクセスしようとするとどうなりますか？
    次の４つから正しい答えを選びなさい。
選択肢：
a. `0`が返される
b. `null`が返される
c. コンパイルエラーが発生する
d. `ArrayIndexOutOfBoundsException`が発生する

答え：
d
解説：
配列のインデックスは`0`から`要素数-1`までです。要素数が10の場合、有効なインデックスは0から9までとなり、`10`は範囲外であるため、実行時に`ArrayIndexOutOfBoundsException`という例外が発生します。
